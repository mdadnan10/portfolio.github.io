snake game

Project brief:

In this project, we propose method for Play Snake game using html and JavaScript language. In this project, you can play the popular Snake Game just like you played it elsewhere. You have to use the up, down, right or left arrows to move the snake. Foods are provided at the several co-ordinates of the screen for the snake to eat. Every time the snake eats the food, its length will by increased by one element along with the score. It isn't the world's greatest game, but it does give you an idea of what you can achieve with a relatively simple JavaScript program, and perhaps the basis by which to extend the principles and create more interesting games of your own.
